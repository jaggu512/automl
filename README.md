# Intelligent AutoML System

A beginner-friendly AutoML application built with Streamlit and PyCaret for automated machine learning tasks.

## Features

- **Easy Project Setup**: Define your ML problem with simple descriptions
- **Dataset Upload**: Support for CSV datasets
- **AutoML Pipeline**: Automatic model training and comparison using PyCaret
- **Explainability**: Feature importance visualization
- **Reports**: Downloadable project reports

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
```

2. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit app:
```bash
streamlit run app.py
```

## Requirements

- Python 3.8+
- Streamlit
- PyCaret
- Pandas
